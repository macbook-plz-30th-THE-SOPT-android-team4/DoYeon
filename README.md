### 구현한 사항
- 로그인 화면 (SignInActivity)
  - 로그인 버튼 클릭시 입력필드 유효성 검증
  - 유효성 검증 성공시 HomeActivity로 이동

- 회원가입 화면 (SignUpActivity)
  - 회원가입 완료 버튼 클릭시 입력필드 유효성 검증
  - 유효성 검증 성공시 이전 화면으로 이동

- 자기소개 화면 (HomeActivity)
  - ImageView, TextView 사용
